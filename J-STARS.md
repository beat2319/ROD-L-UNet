# J-STARS

The J-STARS project focuses on detecting and monitoring Rapid ʻŌhiʻa Death (ROD) using satellite imagery and machine learning.

https://www.grss-ieee.org/wp-content/uploads/2025/09/JSTARS_CfP_Templatev3.pdf
https://github.com/mpapadomanolaki/multi-task-L-UNet

## Project Structure

```
J-STARS/
├── ROD-WEB-APP/          # Web application for ROD detection visualization
├── ROD-ANALYSIS/         # Spatial analysis methods
├── ROD-COLLECTION/        # Data collection processes
├── ROD-PROCESSING/       # Data processing pipelines
└── Documentation/         # Project notes and references
```
